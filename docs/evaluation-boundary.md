# 评测口径与证据边界

## 数据切分

- 总量 N=240。
- 开发/演示集 N=180，用于提示词、规则和聚类阈值开发。
- 锁定合成校验集 N=60，不得用于调整提示词或阈值。
- 当前 N=60 已完成 AI 辅助候选一致性复核；它不是独立人工审计，也不代表真实业务分布。

## 展示指标

面试展示页优先使用：

- 问题类型与产品区域混淆矩阵。
- 每类 precision、recall、F1 和 support。
- 重复问题匹配精确率与召回率。
- 聚类纯度。
- 错误合并与错误拆分案例。
- 代表工单可追溯率。

B³ F1 和阈值搜索只进入技术报告。

| 指标 | 门槛 |

若任一关键门槛未通过，只能声明机制已实现，不能声明分类或聚类质量达标。

## 当前事实

公开评测的权威数值见 `artifacts/evaluation/evaluation.json`。当前为真实 `dify` + BGE、数据版本 `v3-natural-singletons`：问题类型 Macro-F1 0.647、责任路由策略一致率 83.3%，未通过门槛；升级召回、quote 定位和重复问题指标通过。不得把部分通过包装成整体质量达标，也不得把 AI 辅助复核包装成独立人工审计。
